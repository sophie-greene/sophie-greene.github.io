Linux Tips

Find file types

find Downloads/ -type f | perl -ne 'print $1 if m/\.([^.\/]+)$/' | sort -u

Find a specific file type and copy

find Downloads -iname *.epub -exec cp {} /media/some/sophie-hhd/books/epub/ \;

Find a specific file type and copy if not already in distination folder

find Downloads -iname *.epub -exec cp -n {} /media/some/sophie-hhd/books/epub/ \;
