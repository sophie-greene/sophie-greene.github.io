Linux Tips

Find file types 

								
									find / -type f | perl -ne 'print $1 if m/\.([^.\/]+)$/' | sort -u

Find a specific file type and copy 

								
									find / -iname *.epub -exec cp {} epub/ \;

Find a specific file type and copy if not already in destination folder 

								
									find / -iname *.epub -exec cp -n {} epub/ \;
