Add and Use Snippets in VScode

Go to File -> Preferences -> User Snippets . Select your preferred language. Now type the following code to make a for loop snippet: 

					
						"Create for loop":{
							"prefix": "html5",
							"body":[
								"for(int i = 0; i < 10; i++)",
								"{",
								"   //code goes here",
								"}"
							],
							"description": "Creates a for loop"
						}

You are done. Type html5 in the editor and use the first prediction.

Shortcut

  1. Install the snippet generator extension.
  2. Highlight the code that you need to make snippet
  3. pressCTRL + SHIFT + P and type "Create snippet" on the command palette and press ENTER. Alternatively you can right click on the selected code and choose Generate snippet from the popup menu as shown below:
  4. select language for which you want to create snippet(eg:-HTML), then type snippet name, type snippet shortcut and then type snippet description. see below:

Note that if you go to File->Preferences->User Snippets , then select the language in which you created the snippet. You will find the snippet there.

Type the snippet shortcut, that you have chosen in step 4 in the editor and select it from the suggested list. If no suggested list appears, press ctrl + space that comes first.
